# Python Secure Student Registration

## Description

This project is a beginner Python application that securely registers student information using exception handling. It validates user input to ensure only correct data is accepted, making the program more reliable.

## Features

- Register student names, ages, and marks
- Validate numeric input
- Ensure marks are between 0 and 100
- Handle invalid input gracefully
- Register multiple students

## Technologies Used

- Python 3

## Concepts Practiced

- Exception Handling (`try`, `except`, `else`, `finally`)
- User Input
- Loops
- Conditional Statements
- Variables

## Future Improvements

- Save student records to a file
- Search for students
- Edit student information
- Export data to CSV
